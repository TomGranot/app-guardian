Features

Everything your Mac's memory needs

A tight, focused tool. No bloat, no accounts, no subscriptions.

🤖

Apple Intelligence decisions

Uses apfel to ask Apple's on-device Foundation Model: "Should I close this?" Answers are cached per session so it only calls once per app.

🧹

Automatic cache cleanup

After quitting, deletes ~/Library/Caches/<bundle> and any fuzzy-matched folders. Frees space you'd never find manually.

💀

Force-quit fallback

Tries a graceful quit first. If the app doesn't respond within 10 seconds, it force-kills it. Nothing gets left hanging.

🚫

Smart exclusions

System apps (Finder, Dock, Spotlight…) are excluded by default. Add any app to the list directly from the menu.

📈

Usage statistics

Every close event is logged to a local JSON file. The built-in stats screen shows closes today, this week, all time, and top offending apps.

🔒

Fully private

All data stays on your machine. The only network call is to localhost:11434 (apfel). Zero telemetry.

How it works

Simple by design

Three loops running quietly in the background.

1

Poll (every 10 s)

Records which app is in the foreground and timestamps it. Uses macOS's NSWorkspace API — no Accessibility permissions required.

2

Check (every 60 s)

Scans all running apps. Any app idle past your threshold gets flagged. Apple Intelligence (via apfel) makes the final YES/NO call — or you can skip it and use pure timer logic.

3

Close + clean

Sends a terminate signal and waits. Force-kills if needed. Then wipes the app's cache directory and logs the freed space to history.

Installation

Up and running in 60 seconds

# Clone the repo
git clone https://github.com/TomGranot/app-guardian.git
cd app-guardian

# (Optional) Install Apple Intelligence backend
brew tap Arthur-Ficial/tap && brew install apfel
apfel --serve   # keep running in a Terminal tab

# Install Python deps and launch
./setup.sh
./run.sh

# Auto-start on login (optional)
./install-agent.sh
